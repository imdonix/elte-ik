# Mad Tycoon
A real time simulation tycoon game.

## Build tool

To run the project you need: `jdk8` `ant`

```bash
ant run
```

To create a jar file with resources:

```bash
ant jar
```

To create a zip of the build:
```bash
ant zip
```

## IntelliJ IDEA

You can find the `ant` tool in `View -> Tool Windows -> Ant`
